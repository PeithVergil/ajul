== Description ==

This plugin allows you to create a guided tour around your site. A guided tour displays content bubbles that target specific elements on a page.


== Installation ==

1. Upload the `ajul/` directory to the `plugins/` directory of your WordPress installation.
2. Activate the plugin through the **Plugins** menu inside the WordPress Admin.
